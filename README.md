# diffimage

A visual diffs for the images in the git repository.


## Demo
![git diff](images/diff.png)


## Dependencies

Debian/Ubuntu:
```bash
$ sudo apt-get install imagemagick
```

Arch:
```bash
$ sudo pacman -S imagemagick
```

## Installation
```bash
make install
```


## Usage
```bash
git diff
```
